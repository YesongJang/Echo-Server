# Echo Server

A modular network-based Echo server written in Java.

## 💡 Features
- Modular design with handlers:
  - `ProxyHandler`
  - `CacheProxyHandler`
  - `SecurityProxyHandler`
  - `MathHandler`
- Socket-based client-server architecture

## 🛠 Tech Stack
- Java
- Socket programming

## 📁 How to Run

```bash
javac *.java
java ProxyServer
